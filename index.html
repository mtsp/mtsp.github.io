MTSP
The Multicore Task Scheduling Platform


MTSP (Multicore Task Scheduling Platform) is a small, lightweight research runtime aimed at extracting task parallelism by means of OpenMP 4.X annotations. Guided by an extensive profiling study it has been designed for flexibility and easy of use. MTSP is based on OpenMP task parallelism interface and its open API enables runtime designers to fast prototype new task stealing polices and workload balance heuristics.
